# Narbs Tap

## How do I install these formulae?

`brew install narbs/tap/<formula>`

Or `brew tap narbs/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "narbs/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
